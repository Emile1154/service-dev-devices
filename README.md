# Getting started
* You'll need Java 11+ version installed
* go to project directory by command ./cd <= project-space =>
* command ./mvn install
* go to target directory ./cd <= project-space => /target
* run application ./java -jar <= application.jar =>
* go to http://localhost:9000/
